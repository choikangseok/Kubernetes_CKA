### CKA 합격 팁
```
--dry-run=client
$ kubectl create depoly nx --image=nginx --dry-run=client
# --dry-run=client --> 리소스를 생성하지 않음
# 단순한 명령을 테스트하려는 경우에 사용
#사용자의 명령어가 올바른지 알려준다.

$ kubectl create depoly nx --image=nginx --dry-run=client -o yaml
# -o yaml
# 화면에 YAML 형식의 리소스 정의 출력
# 문서를 빨리 만들기 위한 옵션

```
